# powerbi-na-real-materials
Arquivos exclusivos para seguidores do canal Power BI na Real

## Jogos
Este repositório agora inclui um simples jogo de Ping Pong implementado em Python utilizando a biblioteca `pygame`.

### Como jogar
1. Certifique-se de que o pacote `pygame` está instalado (`pip install pygame`).
2. Execute o arquivo `games/ping_pong.py` com Python 3:
   ```bash
   python3 games/ping_pong.py
   ```
3. Use as teclas **W** e **S** para mover a raquete esquerda e as setas **↑** e **↓** para controlar a raquete direita.
